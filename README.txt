
# Legal AI Platform - Complete Project

## 📁 Structure
- `frontend/`: React-based lawyer dashboard with Arabic RTL support.
- `backend/`: Node.js + Express API with SQLite database.

---

## ▶️ How to Run the Full Project

### 1. Backend Setup
```bash
cd backend
npm install
npm start
```
This will run the backend on: http://localhost:3001

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
This will run the frontend on: http://localhost:5173
